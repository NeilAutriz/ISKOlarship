# ISKOlarship

ISKOlarship is a web-based scholarship discovery platform designed to help students find and apply for scholarships that match their profiles. The platform incorporates rule-based filtering and logistic regression to enhance the scholarship search experience.

## Features

- **Scholarship Discovery**: Browse and search for scholarships based on various criteria.
- **Rule-Based Filtering**: Narrow down scholarship options using customizable filters.
- **Logistic Regression Model**: Predict the likelihood of scholarship application success based on historical data.
- **Data Visualization**: Visualize historical scholarship application data and success rates through interactive charts and graphs.
- **User Dashboard**: Personalized recommendations and application status tracking for users.

## Project Structure

The project is organized into the following main directories:

- **src**: Contains all source code, including components, pages, services, data, models, types, utils, hooks, and styles.
- **public**: Contains static files, including the main HTML template.
- **Configuration Files**: Includes `package.json`, `tsconfig.json`, and `vite.config.ts` for project setup and configuration.

## Getting Started

To get started with ISKOlarship, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   cd ISKOlarship
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run the Application**:
   ```bash
   npm run dev
   ```

4. **Open in Browser**:
   Navigate to `http://localhost:3000` to view the application.
