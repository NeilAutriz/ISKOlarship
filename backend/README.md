# ISKOlarship Backend

Backend API for the ISKOlarship scholarship management platform.

## Quick Start

```bash
# Install dependencies
npm install

# Set up environment variables (see .env.example)
# Run the server
node src/server.js
```

## Project Structure

```
src/          # Application source code
tests/        # Test files
scripts/      # Utility scripts (migrations, cleanup, verification)
uploads/      # User uploaded documents
```
