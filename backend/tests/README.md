# Tests

This directory contains test files for the ISKOlarship backend application.

## Test Files

- `test-admin-document-ids.js` - Tests if admin documents have proper _id fields
- `test-admin-document.js` - Tests admin document functionality
- `test-admin-documents-api.js` - Tests admin documents API endpoints
- `test-admin-fix.js` - Tests admin profile fixes
- `test-admin-registration.js` - Tests admin user registration
- `test-clean-data.js` - Tests data cleaning functionality
- `test-data.js` - General data testing
- `test-document-storage.js` - Tests document storage system
- `test-document-upload-optimized.js` - Tests optimized document upload
- `test-document-upload.js` - Tests document upload functionality
- `test-prediction.js` - Tests scholarship prediction service
- `test-profile-integration.js` - Tests profile integration
- `test-scholarship-creation.js` - Tests scholarship creation

## Running Tests

To run a specific test:
```bash
node tests/<test-file-name>.js
```

Example:
```bash
node tests/test-admin-registration.js
```
